# RAG-qwen2.5

Agentic RAG System

An Agentic Retrieval-Augmented Generation (RAG) pipeline that integrates LLM-based agents, semantic retrieval, and automated evaluation using RAGAS.

Features:
- Agent-based query orchestration
- Semantic document retrieval
- Vector similarity search
- LLM response generation
- RAGAS evaluation pipeline
- End-to-end experimentation in Jupyter Notebook
  
Tech Stack:
- Python
- LangChain
- Hugging Face
- FAISS / ChromaDB
- RAGAS
- Jupyter Notebook

Evaluation Metrics:
- Faithfulness
- Answer Relevancy
- Context Precision
- Context Recall
  
Project Goals:
- Explore Agentic RAG architectures
- Improve contextual response generation
- Evaluate retrieval quality using RAGAS
